팀별 메서드 실습 과제

프로젝트명 : 영어 단어 저장 프로그램
1. 영어 단어 5개를 저장할 수 있다.
2. 영어단어저장 기능 : 키보드로 영어 단어를 입력하여 저장한다.
3. 영어단어 전체보기 기능 : 저장된 영어 단어를 모두 출력한다.
4. 영어단어 검색하기 기능 : 키보드로 영어 단어를 입력하면 입력된 단어가 있으면 출력, 없으면 없음이라고 출력

---- 추가 기능
5. 영어단어 삭제 기능 : 키보드로 영어단어를 입력하면 해당 단어를 삭제한다.
6. 영어단어 수정 기능 : 키보드로 영어 단어를 입력하고 해당 단어를 수정할 수 있다.
7. 영어단어 검색 기능 업그레이디 : 키보드로 입력된 단어가 포함이 되면 모두 찾아 준다. 


---갱신
8. 영어단어와 뜻을 모두 입력하여 저장하는단어 프로그램
9. 영어단어 저장 기능 <갱신> : 키보드로 영어 단어와 뜻을 모두 입력한다.
10. 영어단어 전체보기 기능<갱신> : 저장된 영어 단어와 뜻을 보여준다.
11. 영어단어 검색하기 기능<갱신>: 영어 단어로 찾고 영어단어와 뜻을같이 보여준다.

-- 기능 추가
12. 단어시험 기능 : 랜덤한 단어를 3개 뽑는다. 랜덤한 단어의 한글 뜻을 출력하고 단어를 입력하게 하는 프로그램 
                         최종 맞춘 갯수를 출력한다.
13. 기타 추가 하고 싶은 기능들
